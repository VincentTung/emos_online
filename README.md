## 项目介绍

Emos是一个前后端分离的在线办公系统，项目采用 SpringBoot+Uniapp 开发，项目加入常见的企业级应用所涉及到的技术点，例如 Redis、RabbitMQ 等。前端使用Uniapp开发微信小程序，员工可以在微信中通过小程序登录，实现在线办公。



### 项目技术栈

#### 后端技术栈

1. SpringBoot
2. Shiro
3. MyBatis
4. MySQL
5. Redis
6. RabbitMQ
7. MongoDB
8. JWT
9. 腾讯云API(人脸识别 对象存储)
10. ...

#### 前端技术栈

1. Uniapp
2. 微信小程序
3. ...
